# 수열

1. 처음 풀이

   💡아이디어

   ```markdown
   1. dp테이블에 누적합을 저장 
   2. 구하려는 구간의 이전 구간(포함하지 않는 구간) 누적합을 빼기(dp[i+K]-dp[i])
   3. 대소 비교 후 큰 값 저장
   ```

   ✏️풀이

   ```python
   import sys
   input = sys.stdin.readline
   N, K = map(int, input().strip().split())
   temperature_lst = list(map(int, input().strip().split()))
   dp = [0]*(N+1)
   
   for j in range(1, N+1):
       dp[j] = temperature_lst[j-1] + dp[j-1]
   
   max_val = -1e10
   for i in range(N-K+1):
       temp = dp[i+K]-dp[i]
   
       # 1)
       # max_val = max(max_val, temp)
   
       # 2) max 메서드 사용하는 것보다 10ms 정도 빠름
       if temp > max_val:
           max_val = temp
   
   print(max_val)
   ```

   

2. 더 빠른 풀이

   💡아이디어

   ```markdown
   1. k 구간만큼의 합을 두 변수 max_val과 val에 선언 
   2. val에 다음 요소 값을 더하고 더해진 값 중 처음 요소 값을 빼줌 
   3. 대소비교 후 더 크면 값을 업데이트함
   ```

   ✏️풀이

   ```python
   # 10 ~ 20ms 더 빠름
   import sys
   input = sys.stdin.readline
   
   n, k = map(int, input().split())
   a = list(map(int, input().split()))
   
   # 0번째부터 k-1까지 요소의 합 저장
   max_val = sum(a[:k])
   val = sum(a[:k])
   
   for i in range(n - k):
       # ex. a = [1,2,3,4,5,6] 
       # 1+2+3 -> (1+2+3)-1+4 = 2+3+4
       val = val - a[i] + a[k + i]
       if max_val < val:
           max_val = val
   print(max_val)
   ```
