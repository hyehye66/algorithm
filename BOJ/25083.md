# 새싹

[출력]

```python
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```

✏️ 풀이

```python
print('         ,r\'"7')
print('r`-_   ,\'  ,/')
print(' \\. ". L_r\'')
print('   `~\\/')
print('      |')      
print('      |')
```

---

#### ✅출력 방법

`\` 을 출력하려면  `\\` <= 두 번 작성

`""`, 혹은  `''` 안에 같은 것을 출력하려면 `\"`, 혹은 `\'` 으로 작성해야 한다.
