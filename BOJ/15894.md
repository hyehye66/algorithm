# 수학은 체육과목 입니다

```python
import sys
sys.stdin = open('input.txt', 'r')
n = int(input())
# 사각형개수: 1, 1+2, 1+2+3, ...
# 둘레(가로 + 세로): 4, 8=(1*2+2)+2*2, 12=(1*3+3)+2*3, ...
# 둘레 = (가로: 변 1 개가 n 개만큼 있고 제일 아래 밑변의 길이가 n이다) + (세로: 양쪽에 2 개씩 n 개만큼) = 4*n
if n == 1:
    print(4)
else:
    print(1*n+n+2*n)

```
