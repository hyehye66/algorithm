# 왕복

```python
import sys
input = sys.stdin.readline
N, K = map(int, input().strip().split())
distance_lst = list(map(int, input().strip().split()))

total_distance = sum(distance_lst)
temp = K % total_distance
if (K//total_distance)%2:
    for i in range(N-1, -1, -1):
        temp -= distance_lst[i]
        if temp < 0:
            print(i+1)
            break

else:
    for i in range(N):
        temp -= distance_lst[i]
        if temp < 0:
            print(i + 1)
            break
```
